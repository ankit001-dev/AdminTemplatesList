    Hey there, countUp.js is designed for doing short counting animations... maybe from like 1-10 seconds long. If you are doing something like a day counter, you would need to do that using Date(). For instance, to display "X days until Christmas" you would do

    const today = new Date();
const xmas = new Date('2020-12-25T00:00:00Z');
const diff = xmas.getTime() - today.getTime();
const days = diff / (1000* 60 *60 *24);
console.log(`${Math.round(days)} days until Christmas!`);

    Or, maybe you wanted something completely different. In any case, hope that helps.

    I have made a variation so that it shows an incremental updated number, every 2 seconds, based on a preset final number to reach fully on a certain date.

    Final Date = December 1, 2021
    Final Number = 1,000,000

    Between dates examples:
    Oct 1, it will be starting displaying 0, then increase every 2 seconds
    Nov 1 will show 500,000, then increase every 2 seconds
    Nov 15 will show 750,000, then increase every 2 seconds
    Dec 1 will stop at 1,000,000

    Refreshing the page will show the counter starting where it was at, before refreshing.

    
  var counter = 0;
  var loopInt = 2000;
  const finishNumbers = 1000000;
  const startingNumbers = 0;
  var numCount = 0;

  var i = setInterval(function(){
      var numCountDiff = numCount;
      var numDiff = finishNumbers - startingNumbers;
      var todayTime = new Date();
      const endTime = new Date('2021-12-01T00:00:00Z');
      var diffTime = endTime.getTime() - todayTime.getTime();
      var days = diffTime / (1000* 60 *60 *24);
      var diffTimeSeconds = Math.round(diffTime / (1000));
      console.log(`${diffTimeSeconds} seconds (`+Math.round(days).toFixed(2)+` days) until ${finishNumbers}!`);
      numCount = startingNumbers +  ((numDiff/diffTimeSeconds));
      numCountDiff = numCount - numCountDiff;
      console.log(Math.round(numCountDiff).toFixed(2), "nums added");
      counter++;
      if(counter === 100 || ((numCount === finishNumbers)) ) {
          clearInterval(i);
          window.loopfetch( finishNumbers );
      } else {
        window.loopfetch( Math.round(numCount) );
      }
  }, loopInt);

  window.loopfetch=function (evt) {
    console.log("evt = "+evt);
    code += "<br>demo.update(" + evt + ");";
    codeVisualizer.innerHTML = code;
    demo.update(evt);
  };

    The loop, set at 2000 milliseconds, will stop at 100 executions and when it reaches the Final Number on December the 1st.
